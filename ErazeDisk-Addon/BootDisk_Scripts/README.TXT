This is an addition for a tool that cleans a computer before it can safely be removed from an organization.

Cleaning harddrives + factory resetting BIOS + remove BIOS password.
Additionally creating/adding to a .csv file for adding the device to SCCM.

HP specific.
1. A few functions require "HP BIOS Configuration Utility".
2. For password locked BIOS use BiosConfigUtility64.exe to make a .bin passwordfile.
